# Delivery App

App de delivery con web, app móvil y backend escalable en AWS.

## Estructura del proyecto

```
delivery-app/
├── backend/      # API REST con Node.js + Express (desplegable en AWS)
├── web/          # Aplicación web con React
└── mobile/       # App móvil con React Native
```

## Tecnologías

- **Backend:** Node.js + Express
- **Web:** React
- **Mobile:** React Native
- **Infraestructura:** Amazon AWS
